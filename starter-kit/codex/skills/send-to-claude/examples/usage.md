# send-to-claude Usage Examples

## Infrastructure Questions

```bash
/send-to-claude "How should I structure this Supabase edge function?"
```

## Architecture Guidance

```bash
/send-to-claude --type architecture "Should this use RLS policies or app-level permissions?"
```

## n8n Workflow Patterns

```bash
/send-to-claude "What's the right n8n workflow pattern for batch processing?"
```

## Supabase Review

```bash
/send-to-claude --type review "Review this stored procedure for correctness"
```

## With Context

```bash
/send-to-claude --context "Tried queue-based and direct calls" "Which approach is better for this use case?"
```

## Interactive Mode

```bash
/send-to-claude
# Prompts for type, question, context
```
